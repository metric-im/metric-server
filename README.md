# server
Metric IM server for pinging, linking and reporting

## Reserved Attribute Names
* year - YYYY
* monthName - January, February, ...
* month - 1-12
* day - 1-31
* hour - 0-60
* minute - 0-60
* second - 0-60
* week - 1-52
* weekday - Sunday, Monday, ...
* date - YYYY-MM-DD
* _time - now
* _event - event name or "ping"
* _account - authorized account or "public"
